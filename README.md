# LaraShop.io
## Introduction
This is a ecommerce website source code. Build with PHP Laravel Framework.

## Technical used
1. Laravel 5.7
2. AdminLTE
3. VueJs

4. Detail Product
5. Cart
6. Checkout

8. List All Post
9. List Post By Categories
10. Detail Post
12. Page
13. Email Subscribe
14. Membership
